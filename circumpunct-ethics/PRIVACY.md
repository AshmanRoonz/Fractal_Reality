# Privacy Policy

**Circumpunct Ethics Plugin**
**Author:** Ashman Roonz
**Last updated:** March 25, 2026

## Overview

The Circumpunct Ethics plugin is a local reasoning and diagnostic tool. It does not collect, transmit, or store any user data.

## Data Collection

This plugin collects **no data whatsoever**. Specifically:

- No personal information is collected, stored, or transmitted.
- No conversation content leaves your device.
- No analytics, telemetry, or usage tracking of any kind.
- No cookies, identifiers, or fingerprinting.
- No third-party services are contacted.

## How the Plugin Works

All five skills (check, diagnose, detect, steelman, restore) operate entirely within your local Claude session. The plugin provides reference material and structured prompts; it does not make network requests, access external APIs, or store anything between sessions.

## Third-Party Services

None. This plugin has no external dependencies, no MCP servers, and no outbound connections.

## Data Sharing

No data is shared with the plugin author, Anthropic, or any third party through the use of this plugin.

## Changes to This Policy

If this policy changes, the updated version will be published in the plugin repository at https://github.com/AshmanRoonz/Fractal_Reality.
