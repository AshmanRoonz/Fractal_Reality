Six open prediction targets

For each target: the measurement, the framework's expected structural form (what integer pool it must draw from), the experiment whose next result decides it, and the threshold beyond which the framework fails.

1. Muon anomalous moment a_μ

4.2σ tension (SM vs exp)

Fermilab E989 final result: a_μ = 116592055(24) × 10⁻¹¹. SM prediction disputes itself (data-driven vs lattice differ).

a_μ = α/(2π) + framework_correction(α², T, R)

Structural claim: the Schwinger term α/(2π) is dimension-correct (1.5D splitting, same station as lepton mass ratios). Higher-order corrections must draw from {R = 7, V = 13, G = 12}. Framework target: K_gen = T⁴ = 81 as the muon correction denominator (matches m_τ/m_e structure).

Decides it: BMW lattice collaboration final HVP result (2026) or Fermilab's final publication. If experimental central value drifts below framework band, the α²-level integers are wrong.

2. |V_cb| (second CKM angle)

3σ inclusive vs exclusive tension

Measured: 0.0410(14). Inclusive B → X_c l ν and exclusive B → D* l ν disagree.

sin θ_cb = α^(E(2.5)/P + α·c₁) · prefactor

Structural claim: same architecture as Cabibbo (sin θ_C = α^(1/2 + 3α/7) · 8/3). Exponent rises one rung: 1/2 → E(2.5)/P = 56/(39·4) ≈ 0.359. Prefactor from generation-2 to generation-3 mixing: SU(3)/V = 8/13. Naïve framework estimate: α^0.359 · 8/13 ≈ 0.041. Needs the α-correction term fixed by which residual selection rule applies.

Decides it: Belle II precision measurement (expected 2026-2027, targeting 1% precision).

3. Solar neutrino mixing θ₁₂

precision improving

Measured: sin²θ₁₂ = 0.307(13). JUNO will push to ~0.5% by 2027.

sin²θ₁₂ = (Φ+○)/V + framework_correction

Structural claim: leading term should be a simple framework ratio like Weinberg's 3/13. Candidate: (Φ+○)/V = 5/13 = 0.385 (too high), or T/A(2) = 3/10 = 0.300 (close), or A(1.5)/A(2)·(Φ+○)/SU(3) = 6·5/(10·8) = 0.375. Most structurally clean: sin²θ₁₂ ≈ T/A(2) + α·(G/T²) = 0.3 + 0.0081 ≈ 0.308. Agrees with current measurement at 1σ.

Decides it: JUNO first 3-year data (2027). A measurement outside 0.305 ± 0.005 rules out the A(2)-based structural term.

4. Proton charge radius r_p

proton-radius puzzle resolved

CODATA 2022: 0.8414(19) fm. Once-disputed muonic hydrogen value now consensus.

r_p = (λ_Compton,e / α) · (1/α)^(-A(1.5)/P - α·V)

Structural claim: proton radius in electron Compton units: r_p · m_e·c/ℏ = r_p / (386.16 fm) = 0.00218. In α-units: r_p / (α · 386.16 fm) = 0.298. Framework candidate: T/A(2) = 0.300 (matches Weinberg's 3/13 partner ratio). Or: (1/α) × r_p/λ_Compton ≈ 0.298, suggesting a direct framework integer at the boundary station.

Decides it: MUSE collaboration (muon-proton scattering at PSI) final result expected 2026. Sub-percent precision would fix the framework integer or rule it out.

5. W boson mass m_W

CDF vs world avg, 7σ tension

Framework predicts 80,488 MeV (0.15%). PDG avg: 80,369 MeV. CDF 2022: 80,434 MeV. ATLAS 2024: 80,367 MeV.

m_W/m_e = (1/α)^(95/39 − α/Φ)

Structural claim: already locked in at 95/39 = E(2.5)+1 with α/2 correction. Framework central value 80,488 sits between CDF (80,434) and world avg (80,369). The framework is in tension with PDG at 2-3σ and consistent with CDF at 1σ.

Decides it: ATLAS + CMS full Run 3 combination (2026-2027). If the world average moves toward CDF (above 80,430), framework wins; if it stabilizes below 80,400, framework needs a correction term above α¹.

6. Neutron lifetime τ_n

bottle vs beam, 4σ tension

Bottle: 877.8(0.3) s. Beam: 887.7(2.2) s. Methods disagree at 4σ.

τ_n = ℏ/(Γ_β) where Γ_β = G_F² · m_e⁵ · f · (framework prefactor)

Structural claim: lifetime involves G_F (weak Fermi constant) and phase-space factor. G_F in framework terms: G_F = (α/v²) · (T/A(2)) where v is Higgs VEV. Framework pick between the two measurements: bottle result (877.8 s) matches a cleaner integer decomposition. Prediction: true value is 877.7 ± 0.5 s.

Decides it: UCNτ+ experiment at LANL (2026+) targeting 0.1 s precision. Would cleanly resolve bottle-vs-beam and test framework.

§27.7p falsifiable prediction

Every dimensionless coupling in physics, known or future, must decompose as:

coupling = α^k · (framework prefactor from {T, R, P, G, V, SU(3), Φ, ○}) · (1 + α · framework ratio + ...)

where k is the cross-station bond signature. A newly measured coupling that cannot be fit into this decomposition within measurement precision falsifies §27.7p. A coupling that fits supplies its own structural theorem: the ⊂ relation it encodes is the one its signature names. The α-exponent is no longer a number the framework reports; it is an address the framework reads.

Three specific tests

  • Strong coupling at unification: α_s at the GUT scale should equal α (same ladder-distance when scales converge). Measured α_s(M_Z) = 0.1179 runs to ~1/25 at ~10^16 GeV; framework prediction: it meets α_W and α at k=0 convergence.
  • Fourth-generation absence: the K = T^(gen+1) cascade for mass ratios terminates at gen 3 (K = 81). A fourth-generation lepton, if observed, would force K = T⁵ = 243 with specific m_4/m_e prediction; its absence is consistent with framework closure at three generations.
  • Dark matter coupling: if dark matter interacts via a new force, its coupling constant must land at one of the ladder addresses not yet occupied (e.g., 1.5D↔1.5D cross-scale, or 2.5D↔3D). A coupling that does not fit the ladder would be a harder test than non-detection.

What would falsify the framework

Clean kills

  • α drifts outside 0.22 ppb band with measurement precision < 0.1 ppb
  • m_W world average stabilizes below 80,400 MeV (framework predicts 80,488)
  • Any integer in {2, 3, 4, 5, 7, 8, 12, 13, 21, 27, 28, 56, 64} requires refitting across two or more constants
  • T = 3 route fails: a direct measurement forces T ≠ 3 (e.g., nuclear magic numbers with a different R)

Soft warnings (require but do not kill)

  • Higgs quartic λ measurement precision improves and framework 0.10% drifts to >3σ
  • |V_ub| (third CKM angle) refuses to match the 2.5D α-exponent pattern
  • JUNO θ₁₂ outside 0.305 ± 0.005; would force reselection of leading ratio
  • New fourth-generation fermion observed; current K = T^(gen+1) cascade would need extending
