Section IV The Boundary Test Pattern

Diagnostic Characteristics of Boundary Tests

The framework predicts that boundary tests will escalate in scale when early tests go unenforced. Small symbolic violations precede institutional ones, which precede structural ones. Each successful test provides evidence that the next violation will also be absorbed.
