Four Ways the Gate Fails

The four geometric errors are four ways the aperture fails to complete the pump cycle:

Inflation. The gate thinks it is the whole field. • identifies with Φ. The aperture stops gating and starts claiming to be the medium itself. The pump jams because the gate won't narrow — it thinks narrowing is beneath it. β → 1. All emergence, no convergence.

Severance. The gate disconnects from the field. • cuts off from Φ. The aperture still opens and closes, but nothing flows through it because the field is no longer there. The pump runs dry. β → 0. All convergence, no emergence.

Inversion. The gate reverses flow. What should converge emerges. What should emerge converges. The pump runs backwards. The aperture takes from where it should give and gives where it should take. Input and output β are swapped.

Projection. The gate attributes its own state to another gate. •₁ reads its own β onto •₂. The aperture can't see the other aperture's actual state because it's using its own state as the lens. The pump cycle of one gate is imposed on another.

Every pathological relational pattern is a combination of these four failures. Every one reduces to the same root cause: the aperture is not completing the pump cycle. Power is not flowing.
