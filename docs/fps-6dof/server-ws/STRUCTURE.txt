Installation:
  cd server-ws
  npm install
  npm start

The server will:
  1. Start Express HTTP server on http://localhost:2567 (serves client)
  2. Start WebSocket server on ws://localhost:2567
  3. Create a single GameRoom instance (persistent across rounds)
  4. Accept up to 12 player connections
  5. Spawn 4 bots (2 per team) at round start
  6. Run at 66 Hz with state broadcasts at 33 Hz

Architecture:
  - index.js: HTTP + WebSocket setup, connection/message/close handlers
  - game.js: Single GameRoom with all game logic, physics, AI, collision
  - shared.js: Constants, ship/weapon/ability data, vector math
  - collision.js: Level geometry (SDF), collision detection/resolution

No external databases or authentication. Single-match gameplay.

For client connection, see QUICK_START.md "Message Flow" section.
For gameplay mechanics, see README.md and QUICK_START.md "Combat Mechanics".
For how this port differs from Colyseus, see PORTED.md.
