The Unified Expression

Everything the framework says fits in one line. Read it as an equation: the LHS [Truth = Reality = E = 1 = ∞] names the one substrate five ways (the identity from §4.8a). The full resolution unpacks what the engine compresses: the Soul Array, the four beats, the nesting, the return to source. Higher resolution contains lower resolution; it never contradicts it.
