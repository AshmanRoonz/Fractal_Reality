# Circumpunct Core Specification (Minimal Form)

## State Space
```
W = ⊙_space × Φ_space              // paired whole-field states
[⊙⇄Φ] ⊂ W                          // coupled states only
```

## Coupling
```
⊙ = F(Φ)    Φ = G(⊙)               // mutual determination
⊙ = F(G(⊙)) Φ = G(F(Φ))            // fixed points
```

## Energy Flow
```
E : W → W                           // state transformation
E = ⊰ ∘ i• ∘ ≻ ∘ ⊰ ∘ i○^∞ ∘ ≻      // internal decomposition
```

## Compact Form
```
[⊙⇄Φ] ≻ i○^∞ ⊰ •^∞ ≻ i• ⊰ [⊙'⇄Φ']
  └──────────────────────────────┘
    energy pulse through system
    while pair stays coupled
```

## Physical Interpretation
- `i○^∞` = infinite boundary apertures (interface with environment)
- `•^∞`  = infinite power lines (1.5D structure, the "golden pegs")
- `i•`   = center aperture (convergence and transformation point)

## Conservation
```
Energy([⊙⇄Φ]) = Energy([⊙'⇄Φ'])
dE_⊙/dt = -dE_Φ/dt                  // energy exchange between components
```
