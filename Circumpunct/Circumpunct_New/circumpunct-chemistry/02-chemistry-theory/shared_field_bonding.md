## THE β = 0.5 PRINCIPLE IN BONDING

### Balanced Sharing

```
For a covalent bond between identical atoms:

   β = (contribution from •_A) / (total from both •'s)
   
   SYMMETRIC BOND: β = 0.5
      Equal sharing, maximum stability
      Example: H-H, C-C, N-N
```

### Polar Bonds

```
For a bond between different atoms:

   β ≠ 0.5
   
   Example: H-Cl
      Cl is more electronegative
      β_H ≈ 0.3, β_Cl ≈ 0.7
      
   Still bonded, but ASYMMETRIC sharing
   Field Φ_shared biased toward Cl nucleus
```

### Ionic "Bonding"

```
Extreme case: β → 0 or β → 1

   Example: Na-Cl
      β_Na ≈ 0.05, β_Cl ≈ 0.95
      
   Almost complete transfer:
      Na → Na⁺ + e⁻
      Cl + e⁻ → Cl⁻
      
   Field Φ essentially belongs to Cl
   Na and Cl held together by Coulomb attraction,
   not truly "shared" field
```
