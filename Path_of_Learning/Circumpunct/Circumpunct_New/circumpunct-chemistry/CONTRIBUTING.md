# Contributing to Circumpunct Chemistry

Thank you for your interest in contributing to the Circumpunct Chemistry project! This framework is open for academic collaboration and we welcome contributions from researchers, developers, and enthusiasts.

- Implement additional molecules in the compiler
- Add visualization tools

### 4. Code Improvements

## üìù Guidelines

### Scientific Integrity
- All claims must be backed by evidence
- Report failures as well as successes
- Clearly distinguish predictions from fits
- Maintain honest assessment of limitations

### Code Quality
- Follow existing code structure
- Add docstrings and comments
- Include test cases
- Update documentation

### Academic Standards
- Cite sources appropriately
- Acknowledge collaborators
- Maintain peer review standards
- Share data and methods openly

## üí¨ Communication

### Reporting Issues
- Describe what you expected vs what happened
- Include code snippets and error messages
- Specify your environment (Python version, OS)
- Share relevant data files

### Proposing Features
- Explain the scientific motivation
- Describe expected behavior
- Suggest implementation approach
- Discuss potential challenges

### Academic Collaboration
- Contact via repository for research partnerships
- Co-authorship opportunities for significant contributions
- Joint grant applications possible
- Conference presentation collaborations

## üî¨ Research Ethics

### We Are Committed To:
- Open science and data sharing
- Reproducible research
- Honest reporting
- Credit to prior work
- Collaborative development

### We Avoid:
- Parameter fitting disguised as prediction
- Selective reporting of results
- Overclaiming accuracy
- Ignoring contradictory evidence
- Unfounded extrapolations
